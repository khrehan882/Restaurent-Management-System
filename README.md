# 🏨 Hostel Management System (FYP)

A comprehensive Hostel Management System developed as a Final Year Project (FYP) to automate and simplify hostel administration tasks. The system provides an efficient platform for managing students, rooms, hostel records, allocations, complaints, and administrative operations.

## 📌 Overview

Traditional hostel management processes often rely on manual record keeping, which can lead to inefficiencies, data inconsistencies, and increased administrative workload. This project digitizes hostel operations by providing a centralized management system for students and administrators.

The system aims to improve operational efficiency, reduce paperwork, and provide quick access to hostel-related information.

## 🚀 Installation Guide

### Prerequisites

* XAMPP or WAMP Server
* PHP 7.x or above
* MySQL
* Web Browser

### Setup Steps

1. Clone the repository:

```bash
git clone https://github.com/khrehan882/Hostel-Management-System-FYP-.git
```

2. Move the project folder to:

```text
xampp/htdocs/
```

3. Start:

   * Apache
   * MySQL

4. Open phpMyAdmin:

```text
http://localhost/phpmyadmin
```

5. Create a new database.

6. Import the provided SQL file.

7. Open the project:

```text
http://localhost/Hostel-Management-System-FYP-
```
